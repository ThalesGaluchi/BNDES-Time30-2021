## Instruções - Para usuário Linux

Abra uma janela de comando e  execute os comandos abaixo:

1) Instale pip:
sudo apt-get install python3-pip

2)Instale o virtualenv:
sudo pip3 install virtualenv 

3) Clone o repositório:


4) Crie e ative um ambiente virtual conda:

conda create -n venv python=3.7.6
conda activate venv

5) Instale os requerimentos:		
pip install -r requirements.txt

6) Rode o app:				
python app.py

7) Acesse pelo navegador (Chrome, Firefox ou Edge):	
http://127.0.0.1:8050/
